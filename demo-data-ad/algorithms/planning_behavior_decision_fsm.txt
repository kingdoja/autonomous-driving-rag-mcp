# FSM 行为决策算法

## 算法概述

基于有限状态机 (FSM) 的行为决策算法，通过状态转换实现自动驾驶车辆的高层决策。

## 状态定义

### 主要状态
1. **CRUISE**: 巡航
2. **FOLLOW**: 跟车
3. **OVERTAKE**: 超车
4. **LANE_CHANGE_LEFT**: 左变道
5. **LANE_CHANGE_RIGHT**: 右变道
6. **STOP**: 停车
7. **EMERGENCY_STOP**: 紧急停车

## 状态转换

### 转换条件
```
CRUISE → FOLLOW: 前方有车 && 距离 < safe_distance
FOLLOW → OVERTAKE: 可超车 && 左车道空闲
OVERTAKE → CRUISE: 超车完成
CRUISE → LANE_CHANGE_LEFT: 需要左转 || 左车道更优
ANY → EMERGENCY_STOP: 紧急情况
```

### 优先级
```
EMERGENCY_STOP > STOP > LANE_CHANGE > OVERTAKE > FOLLOW > CRUISE
```

## 决策逻辑

### 跟车决策
```python
if front_vehicle_detected:
    distance = get_distance(front_vehicle)
    if distance < safe_distance:
        return FOLLOW
    else:
        return CRUISE
```

### 变道决策
```python
if lane_change_needed:
    if target_lane_safe and gap_sufficient:
        return LANE_CHANGE_LEFT/RIGHT
    else:
        return FOLLOW
```

### 超车决策
```python
if front_vehicle_slow and overtake_allowed:
    if left_lane_clear and oncoming_safe:
        return OVERTAKE
    else:
        return FOLLOW
```

## 代码示例

```python
class FSMBehaviorPlanner:
    def __init__(self):
        self.state = State.CRUISE
    
    def update(self, perception, prediction):
        # 状态转换逻辑
        if self.state == State.CRUISE:
            if self.has_front_vehicle(perception):
                self.state = State.FOLLOW
        
        elif self.state == State.FOLLOW:
            if self.can_overtake(perception, prediction):
                self.state = State.OVERTAKE
        
        # 返回行为指令
        return self.get_behavior_command()
```
