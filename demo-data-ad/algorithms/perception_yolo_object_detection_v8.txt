# YOLOv8 目标检测算法 - 自动驾驶应用

## 算法概述

YOLOv8 是 Ultralytics 公司推出的最新一代实时目标检测算法，在自动驾驶场景中实现了速度和精度的最佳平衡。

## 网络架构

### Backbone: CSPDarknet
- **输入**: 640×640 RGB 图像
- **特征提取**: C2f 模块（改进的 CSP Bottleneck）
- **下采样**: 5 个阶段 (P1-P5)
- **输出**: 多尺度特征图

### Neck: PAN-FPN
- **自顶向下路径**: 特征金字塔
- **自底向上路径**: 路径聚合
- **特征融合**: Concat + C2f

### Head: Decoupled Head
- **分类头**: 类别预测
- **回归头**: 边界框预测
- **Anchor-free**: 无需预定义锚框

## 核心特性

### 1. 高效特征提取
- **C2f 模块**: 更快的梯度流
- **SPPF**: 空间金字塔池化
- **注意力机制**: 可选 CBAM/SE

### 2. 损失函数
- **分类损失**: BCE Loss
- **回归损失**: CIoU Loss + DFL
- **权重**: 动态任务对齐

### 3. 数据增强
- **Mosaic**: 4 图拼接
- **MixUp**: 图像混合
- **HSV**: 色彩空间增强
- **随机翻转/缩放/裁剪**

## 代码示例

### 推理代码
```python
from ultralytics import YOLO

# 加载模型
model = YOLO('yolov8s.pt')

# 推理
results = model('image.jpg', conf=0.25, iou=0.45)

# 解析结果
for r in results:
    boxes = r.boxes  # 边界框
    for box in boxes:
        cls = int(box.cls[0])  # 类别
        conf = float(box.conf[0])  # 置信度
        xyxy = box.xyxy[0].tolist()  # 坐标
        print(f"Class: {cls}, Conf: {conf:.2f}, Box: {xyxy}")
```

### 训练代码
```python
from ultralytics import YOLO

# 加载预训练模型
model = YOLO('yolov8s.pt')

# 训练
model.train(
    data='bdd100k.yaml',
    epochs=300,
    imgsz=640,
    batch=64,
    device=0
)
```

## 常见问题

### Q1: 小目标检测效果差？
**解决**:
- 增加输入分辨率（640 → 1280）
- 使用多尺度训练
- 增加小目标样本

### Q2: 远距离目标漏检？
**解决**:
- 调整 anchor 尺寸
- 增加远距离样本
- 使用级联检测

### Q3: 实时性不足？
**解决**:
- 使用 YOLOv8n（nano 版本）
- 模型量化（FP16/INT8）
- 降低输入分辨率
